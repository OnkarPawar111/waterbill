Water Bill Calculator (HTML, CSS, JavaScript)

This is a simple Water Bill Calculator made using HTML, CSS, and JavaScript.
The user enters the number of water units consumed, and the program calculates the total bill based on slab rates.

Billing Rules

1–100 units → ₹1.25 per unit

101–200 units → ₹2.50 per unit

201–300 units → ₹3.75 per unit

301–400 units → ₹5.00 per unit

Above 400 units → ₹10.00 per unit

Features

Simple and clean user interface

Automatic bill calculation

Input validation

Reset button to clear data

This project is useful for beginners to understand JavaScript functions, conditions, and DOM manipulation.
